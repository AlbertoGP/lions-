24.8 The DL11/KL11 Terminal Device Handler

The file “kl.c” constitutes the device handler for terminals connected to the system via DL11/KL11 interfaces. This group always has at least one member – the operator’s console terminal. Hence this device handler will always be present.

Each DL11/KL11 hardware controller provldes an asynchronous, serial interface to connect a single terminal to a PDP 11 system. For more complete details regarding this interface the reader should consult the “PDP11 Peripherals Handbook”.
