Other Files in Section One

main.c
[Sheets 15..17; Chapters 6, 7] contains ``main'' which performs various initialisation tasks to get UNIX running. It also contains ``sureg'' and ``estabur'' which set the user segmentation registers.

slp.c
[Sheets 18..22; Chapters 6, 7, 8, 14] contains the major procedures required for process management including ``newproc'', ``sched'', ``sleep'' and ``swtch''.

prf.c
[Sheets 23, 24; Chapter 5] contains ``panic'' and a number of other procedures which provide a simple mechanism for displaying initialisation messages and error messages to the operator.

malloc.c
[Sheet 25; Chapter 5] contains ``malloc'' and ``mfree'' which are used to manage memory resources.
